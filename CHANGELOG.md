## [2025-03-09] 设定与执行分离

- **场景设定**：仅含场景名、计划日期、备注、服务分类
- **执行记录**：独立数据，记录是否触发、触发时间、地点、具体情况、图片
- **执行表单**：支持「已触发/未触发」选择，触发时填写时间与地点
- **场景详情**：展示设定 + 执行记录列表，可添加/编辑/删除执行
- **首页/列表**：今日计划、即将执行，「记录执行」快速入口
- **导入导出**：场景与执行一并导出/导入，版本升级至 v2

## [2025-03-09] 预设场景一键导入

- 新增 17 个 MaaS 测试场景预设（公共交通 6、动态交通 5、感知辅助 6）
- 设置页「导入预设场景」按钮，合并模式一键导入
- `scenarios-seed.json` 部署后可访问

## [2025-03-09] GitHub Pages 自动部署

- 新增 `.github/workflows/deploy-pages.yml`，push 到 main/master 自动构建部署
- 需在仓库 Settings → Pages 将 Source 改为 **GitHub Actions**
